# skills 🧰✨

一个用来存放我自己常用 skills 的小仓库。

这里的 skill 来源比较随缘但很务实：
- 有一些是我自己写的 🛠️
- 有一些是在别的 skill 市场里淘来的 🔎
- 也有一些是在原有 skill 基础上，按自己的使用习惯改过的 🔧

## 这个仓库主要想做什么

目标很简单：**沉淀一批真正常用、通用、实用的 skill**。

我比较偏好的方向是：
- **通用能力**：不要太依赖某个特别窄的场景，尽量能跨项目复用
- **实用能力**：不是“看起来很厉害”，而是真的能提高效率
- **即拿即用**：尽量少折腾配置，拿来就能帮上忙
- **持续迭代**：好用就留下，不顺手就继续改

换句话说，这里不追求 skill 数量很多，比较追求的是：

> 放进来的东西，最好是真的会反复用到。🙂

## 仓库里会放什么

这里一般会包含：
- 日常开发里高频会用到的 skill
- 能提升总结、分析、检索、表达效率的 skill
- 能减少重复劳动的 skill
- 经过自己实践后，觉得值得长期保留的 skill

## skill 来源说明

这个仓库不是“全部原创作品展”。

更准确地说，它是一个 **个人常用 skill 集合**：
- 有原创
- 有改造
- 有收藏
- 有二次整理

所以我会更关注：
- 这个 skill 是否好用
- 是否稳定
- 是否足够通用
- 是否值得长期保留

## 欢迎一起塞点好东西进来 🚀

如果你也有觉得很不错的 skill，欢迎：
- 提 PR
- push 你觉得真的好用的 skill
- 分享你改造过、优化过的版本

尤其欢迎这几类：
- 通用性强的
- 实战里确实有帮助的
- 表达清晰、规则明确的
- 拿来就能用、不容易跑偏的

如果一个 skill 能让人发出一句：

> “这个我下次还想继续用。”

那它就很适合放进这里 😄

## 一个很朴素的收录标准

不会特别复杂，大概就看这些：
- **有没有用**
- **会不会常用**
- **够不够稳**
- **别人拿去也能不能直接受益**

## 最后

这个仓库本质上就是一个持续生长的 skill 小工具箱 🧳

希望它慢慢变成一个：
- 对我自己足够顺手
- 对别人也有点参考价值
- 打开时不会眼花缭乱，但每个 skill 都挺能打

欢迎来玩，欢迎贡献，也欢迎把你觉得好的 skill 一起放进来。🙌
